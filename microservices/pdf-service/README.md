# calendar-pdfservice

## Docstring Convention

Use Sphinx-style docstrings for Python code in this package.

- For classes, document purpose plus main attributes with ``:ivar`` and ``:vartype``.
- For functions and methods, use ``:param``, ``:type``, ``:return``, and ``:rtype`` where relevant.
- Keep docstrings focused on behavior and contract, not implementation trivia.
